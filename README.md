# 🤖 ОГЭ Информатика Бот

Telegram-бот для подготовки к ОГЭ по информатике. Проверяет решения задач на Python.

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `/start` | Начать работу |
| `/tasks` | Список заданий |
| `/task <номер>` | Взять задание |
| `/hint` | Подсказка |
| `/help` | Помощь |

## 🛠 Технологии

- Python 3
- pyTelegramBotAPI
- Запуск кода в песочнице

## 🏃‍♂️ Запуск локально

```bash
pip install -r requirements.txt
echo "TELEGRAM_TOKEN=твой_токен" > .env
python bot.py
